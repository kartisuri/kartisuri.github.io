Autonomous Drone

An autonomous drone that can fly on its own if we set way points on the map using dedicated mobile application running on android. It is used to take pictures of large objects and send back the images to mobile station (android application) in real time that is used for 3D scanning. It is integrated with ultrasonic sensors for obstacle detection and indoor navigation. I also developed a custom android application that acts as a mobile station for the drone. Unlike conventional methods like WIFI, Radio Telemetry, this drone can be controlled by 4G LTE networks for extended coverage.
